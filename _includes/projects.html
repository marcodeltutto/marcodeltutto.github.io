Virtual Reality Tutorial 1

Tutorial on how to get started with the HTC Vive and Unity.

This tutorial has been put together for the Oxford VR/AR Hub Summer School, held in Oxford in June 2018.

In this tutorial you will learn:

  • How to setup a scene in Unity with interface to the HTC Vive
  • How to use the controlles input and interact with objects in the scene
  • How to create a laser pointer
  • How to teleport in the scene
